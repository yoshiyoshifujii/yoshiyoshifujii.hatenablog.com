久しぶりに投稿。

Ubuntu 12.04 LTS にCUIでDropboxをインストールしたので、メモ。

正確にはVNCも使ったけど…。

+[https://www.dropbox.com/install?os=lnx:title] からubuntu用のdebファイルをDL
+以下のコマンドを実行

>|zsh|
sudo dpkg -i dropbox_1.4.0_i386.deb
||<

+次にvncで接続してから、以下のコマンドを実行

>|zsh|
dropbox start -i
||<

+するとDropboxのインストールが開始されるので画面表示にしたがう

以上でインストール完了。

VNCのインストール方法については、

http://www.uetyi.mydns.jp/wordpress/colinux-setting/entry-279.html

を参考にさせていただいた。ありがとうございました。
