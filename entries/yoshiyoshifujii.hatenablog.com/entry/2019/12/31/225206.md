ふりかえってみます。

# 1月

Scala福岡2019で登壇させていただきました。

[https://speakerdeck.com/yoshiyoshifujii/shi-jian-clean-architecture:embed:cite]

Scala福岡は、2017年に初参加させていただき、福岡がサイコーだったのもあり、今年は絶対登壇したいと思っていて、CfPを当選いただき、人生2度目の福岡に馳せ参じさせていただいたのでした。満喫しすぎた様子は以下のブログにまとめていました。

[https://yoshiyoshifujii.hatenablog.com/entry/2019/01/26/093926:embed:cite]

2月以降が以下に続きます。

# 2月

社内の重要なプロジェクトのリードをさせていただく機会をいただきまして、そのプロジェクトの立ち上げの月でした。また、サマーインターンシップの企画をし立ち上げだした月でもあります。さらにプライベートでは、家を建てるために土地を購入した月だったりします。今年の2月は、かなり大きく動き出した月となっていました。公私共にすごい動いた月だったんですね。

# 3月

2月に立ち上げたプロジェクトをリードしていくにあたり、隔週で東京に出張させていただき、いろいろとチームビルディングの仕掛けをさせていただいたり、情報を集めてどう進めていくか何故するのか何をするのかといったあたりをあーだこーだと激しく議論する機会を持たせていただきました。この月は兎に角、情報収集をしてそれを溶き解して、理解をしながら筋道をたてて、論理的な抜け漏れがないか他の観点は無いか、納得と説得ができるかどうかなどなど、色々なことに取り組んでいました。ロジカルシンキングのMECEとか、よいイシューの3条件とか、ゴールを設けること、それを満たす主要成功要因を挙げること、KPIを持つことなどなど…。今まで取り組んでいなかったことに取り組ませていただく機会を得ることができました。3月にインプットに必要として購入したり読み直したりしていたのは以下の書籍です。

# 4月

3月と同じ感じで隔週で東京出張してプロジェクトのリードに取り組んでました。

# 5月

令和になりましたね。GW中におっさん会したり地鎮祭したり週末は新築の打ち合わせしたり隔週で東京出張したりとしていました。今年はジムやボルダリングにアクティブに取り組んでおり、出張のたんびにボルダリングに行かせていただいていました。

# 6月

初めての登山に小野アルプスに行きました。今年は、ジムとボルダリングと登山に目覚めた年です。その記念すべき一回目がこの6月でした。いま思えば、もっと春先からはじめていれば良かったなーと思います。一番良い季節のときに行けておらず、夏が厳しくなるころに山に行きはじめましたので、もっと年間通して行きたかったなーっていうのが正直な感想です。が、山は人生の趣味として続けられる気がしていますので、気長に慌てず楽しんでいけると良いなーって思います。また、6月の末にScalaMatsuri2019がありました。2年ぶり3回目の登壇をさせていただきました。また、所属している会社のブースを出す機会も得ることができ、Scala酒場なるScala関西Summitのアンカンファレンスで埋まれたコンテンツを何故か弊社のブースでさせていただくという機会も得て、かなり楽しかった思い出です。ありがとうございました。また、おっさん会で、ディズニシー飲みをしたり、帰りに小田原のサイコーの回転寿司をいただいたりと、なんともたまらん月末でした。

[https://speakerdeck.com/yoshiyoshifujii/clean-architecture-in-practice-at-scalamatsuri2019:embed:cite]

# 7月

上棟式をしたり週末に山登りにいったその後に大阪の勉強会に参加したりとプライベートでやたらと山に行きました。隙あらば山に行ってました。8月になったら暑くて登れないだろうと思っていたので隙あらば行きたかったのです。仕事のほーでは、大阪のオフィスが移転しましたので、移転パーティとかあってかなり楽しかった思い出があります。

# 8月

2年連続2回目のライジングサンですね。去年に引き続き台風とおいかけっこをして、今回は1日目が中止になるというかなり最悪なことになり…1日、札幌でやることもなくボルダリングに行ったり映画を観たりと、家におっても変わらん過ごし方を札幌でしてましたねー。とはいえ、ライジングサンはやっぱりサイコーでした。2回目なのでどう過ごすといいのか、かなり流れも分かっていましたし、良い感じに疲れ過ぎないように過ごすことができました。ただ、2日間ないと少し物足りなかったです。とはいえ、札幌の夜は今年もサイコーでした。たのしかったー。仕事のほーでは、ライジングサンから帰ってきて休む間もなくサマーインターンシップがはじまりました。シンプルに旅の疲れが残っている状態で、会社として初めての取り組みにトライさせていただきました。結果的にこの取り組みを年末の全社表彰の機会でナイスなプロジェクトでしたってことで表彰していただくことができました。この取り組みに関わってくださった皆様に感謝です。

# 9月

所属している会社が上場しました。上場する機会にちょうど所属できてるって中々ない体験だと思いますので、すごい機会に恵まれたなーと感謝です。私自身、ちょうど1年前の9月にジョインしましたので、上場にそこまで貢献できていないのですが、今後の株価に影響を与えられるような働きをしていけたらなーと思っています。またプライベートでは引越ししたり、夏の暑さがやわらいだので登山にいったりしていました。引越しするのが人生で4回目ぐらいではありますが、前回引越しをしてから10年ぐらい経っておりますので、かなり不要な荷物があります。とにかく不要なモノを捨てるため、11月に引っ越すのですが、2ヶ月前から動きださないと絶対に間に合わないと考えていました。山に行きたいけど引越しもしないとだしと週末の過ごし方がかなりやきもきしていました。

# 10月

Scala関西Summit2019ですね。今年まで牽引されてきた偉大なきの子さんが主催から抜けられたこともあり、運営体制を大きく変更しての開催となりました。開催にあたりまして、誰が主催するのか曖昧な状態ではありましたが、スタッフのみなさんが、色々なシーンでそれぞれがリーダーシップを発揮されて、とてもバランス良く運営できたなーと思っています。すごいバランスで進めることができて、ほんとうに良かったなーと思います。なんせ今年も楽しかったです。来年はどうなるかなー。年明け早速、話し合いですねー。

# 11月

新居に引越しました。が、引越した翌日から東京出張、毎週はいりました。これはもしや…家を買うと飛ばされる例のやつか！と戦々恐々としました。が、なんとか、単身赴任ではなく、毎週出張程度におさまっています。2月から立ち上げてリードしてきていたプロジェクトが、次のステージに行くため、その立ち上げに出張が必要なワークとなり東京出張が毎週必要となった感じです。新居に週末しか泊まらない日々が続きましたが、東京でするワークがかなり勉強になりました。なんせJIGはいいぞ。オブジェクト指向たのしいぞ。ってなってます。

# 12月

そんなこんなで毎週出張しながら新居も落ち着いてきて、Advent Calendarも3件投稿できて、最後の全社会議で下半期MVPの表彰をいただくことができました。すごい。MVPみたいな特別表彰されるようなことって、人生で今まで一度もなかったので、驚きがハンパなかったです。2月立ち上げたプロジェクトのリードとサマーインターンシップのリードを評価いただけたとのことで、めちゃ嬉しいです。来年も貢献できるようにがんばります。あと、プライベートではなんといってもスターウォーズ エピソード9ですね。42年が完結しましたね。公開日の0時の会を観て、4DX吹替えで観て、普通に観て、3回観に行きました。なんせサイコーです。エピソード7を観たときに、エピソード9を観るまで生きようと心に誓ったのを思い出します。頑張って生きてきて良かったです。

# まとめ

ざっと書くと色々あったなーと思います。公私共に今年ほど充実した一年はなかったんじゃないかなーと思います。来年も趣味に仕事に色々と取り組む日々を送れるよう健康に気を付けていけたらなーと思う次第です。

本年もお世話になりました。良い御年をお過ごしください。来年もよろしくお願いいたします。
