根気よくダイアリーを続けるため、hatena.vimを入れて、VIMではてダラできるようにした。

その作業内容を早速、hatena.vimから投稿する。

**環境

-Ubuntu 10.04 LTS
-Vim 7.2

**プラグインを取得する

Gitでローカルにcloneする

>||
$ cd ~/work
$ git clone git://github.com/motemen/hatena-vim.git
||<

これで、~/work配下に、hatena-vimが出来る。取ってきたプラグインをコピーする。

>||
$ mkdir ~/.vim/vimfiles/hatena
$ cp -R ~/work/hatena-vim/* ~/.vim/vimfiles/hatena/
||<

**.vimrcに書く

以下のような記述を書く

>|vim|
" for Hatena.vim
set runtimepath+=$HOME/.vim/vimfiles/hatena
let g:hatena_user = "yoshiyoshifujii"
||<

これで終わり。

使い方は、~/.vim/vimfiles/hatena/plugin/hatena.vimを見れば日本語で書いてあるから、分かるはず。
