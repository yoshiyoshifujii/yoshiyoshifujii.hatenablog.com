# あじぇんだ

1. ちょっと座学
2. さっそくやってみよ
3. 感想いおう

って感じでやりました。

# 1. ちょっと座学

[http://yoshiyoshifujii.hatenablog.com/entry/2017/04/19/172606:title] に書いた内容をベースに、Management 1.0, 2.0, 3.0のそれぞれについて、簡単に紹介させていただき、

* うちってまさしく2.0じゃん
* これって幹部の皆さんにも知ってもらいたい!! 

みたいな感想をいただきました。

※ 別に2.0が悪いってわけじゃないですが、3.0があるなら最新にバージョンアップしたいよねって感じです。

# 2. さっそくやってみよう

[ムービングモチベーターズ](https://1qjpt15fhlq3xjfpm2utibj1-wpengine.netdna-ssl.com/wp-content/uploads/2017/03/Management30-MovingMotivatorCards-2015-self-print-A4-Japan.pdf) を実際に、参加者の皆さんにやってもらいました。

[f:id:yoshiyoshifujii:20170418123609j:plain]

# 3. 感想いおう

やってみた感想をシェアしていただきましたところ、

* 相手のことが良くわかる
* やっぱりそうかっていうのと、逆にそうなんだ!って気付きが得れて楽しい
* 順位を聞くだけでも、その人っぽいとか、意外だね!!ってなる
* ぜひ今のプロジェクトや組織でやってみたい!!

といった好意的な感想をシェアいただくことができました。

# 最後にフィードバック

やってみたフィードバックを得たかったので、 Management 3.0 のツールで、 [Happiness Door](https://management30.com/practice/happiness-door/) というのがあります。

これをやってみて、フィードバックをいただきました。

[f:id:yoshiyoshifujii:20170418125512j:plain]

帰り際に、感想をポストイットに書いていただいて、感情の4段階のうち、どうだったか貼ってもらいます。

それだけで、この会に対して、どういう評価になったのか一目瞭然で分かります。

ポストイットには感想を書くも良し。書かずにHappy度合いを貼るだけでも良し。

楽しくフィードバックをしていただけますし、上のほーにいっぱい付箋が貼ってあったら、やった側もHappyになりますね。

下のほーにいっぱいあったら、それはそれで、改善のチャンスですから、気をつかわず、思いをドアにブツけて帰っていただければと思いますね。

# まとめ

皆さんから高評価をいただけて、とてもやって良かったなーと思いました。

やっぱ体験すると、どっかで紹介したり、自分も導入してみたり、試してみようって気になりますよね。

人に話したくもなるし、話しても嘘じゃないリアルな体験と自分なりの所感を持てるので良いですね。

ぶっつけ本番は誰しも怖いでしょうから、組織やチームで試す前に、こーやってお昼にゆるーく集まって素振りするの、良いんじゃないかな。

トップダウンじゃなくて、地味にこーやって広めていっていけば、Management 3.0が会社組織に根付く日が来るのやも??

以上です。
