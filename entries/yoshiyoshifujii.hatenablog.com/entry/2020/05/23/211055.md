今までマシンを自作したことがなかったのですが、どうしても強いマシンが欲しくて一念発起してマシンを自作した。

備忘録的にどんなパーツで組んだか残しておきたいので、書く。

以下、購入したパーツをずらっとご紹介。

<!-- more -->

# CPU

[asin:B005404P9I:detail]

この記事を書いているときは、第10世代インテルのCPUが発売されていますが、今回は、第9世代のCore i9 9900Kを購入した。

8コア16スレッド。

# マザーボード

[asin:B07J53J483:detail]

安定のASUSにしました。

↑のリンクはちょっとタイトルが違いますが、 ASUS ROG STRIX Z390-F GAMING【ATX】を購入した。

世代交代の直前ということもあり、Z390系のマザーボードは入手し辛い時期でしたが、お値段的にそんなに上がっていない様子だったのと、オーバクロックも対応しているとのことで、ちょうど良かった。

# メモリ

[asin:B0874BJFMW:detail]

メモリは、64GBは最低で欲しかったので、32GBが2枚で、価格的に抑えめのやつを選んだ。

Samsungのメモリよりちょっと価格帯がお手頃だったので、こちらにした。

# SSD

[asin:B07SD442PS:detail]

1TBぐらいあれば十分だなーってことで選んだ。

# クーラー

[asin:B0723HNWCW:detail]

どの程度冷やしたら良いのか分からないけど、空冷はちょっと心許ないかもと思い、水冷で2機クーラー付いているぐらいでいってみた。

# 電源

[asin:B07D2KNCXD:detail]

電源も善し悪しがよく分かってないのもあって、お値段的に特にこれでいっかって程度で選んだ。

買ったあとに知ったが、電源はGOLDとかプラチナとかレベルがあるみたいで、GOLD以上にしておくのがオススメということで、ギリだったみたい。

# グラフィックボード

[asin:B07PCCS63W:detail]

そんなにバリバリゲームをすることも、そんなに無いので、グラボもどの程度のものが良いのかピンときていなかったが、コストを少しおさえめで無難なラインで選んだ。

# ケース

[asin:B078STD6GD:detail]

ケースもサイズ感とか、手の入れやすさとか、よー分からんなーってなってたけど、とりあえず、マザーボードが入りつつ予算的に収まりそうなものを適当に選んだ。

ケースとかそんなに買い替えるものでもないとか色々情報を得ていたが、トータルの予算で考えて無難なあたりにした。

# Wifi & Bluetooth

[asin:B07VCCZS54:detail]

Wifiが欲しいというより、Bluetoothのキーボードやマウスをつなぎたいので、購入した。

# まとめ

ざっと、以上の構成を組んだ。

はじめての自作ということで、用意するモノの選定はけっこー悩んだし、いろいろと調べた。

けど、全部のパーツについて調べ切ることはできなくて、最後はエイヤーって感じで決めたモノも多いけど、次はもちょい構成を工夫できるかも。

あと、組むときも、段取りが分かっていないので、付けたあとにコードが届かなくて外したり、マザーボードに先にメモリとか挿しておかないといけないのに、クーラー取り付けてもて、外して付けてみたいなこと何回もしたり…

ケースのUSBとか使う予定ないのに配線して苦労したり…

まあ、いろいろあったけど、それなりに組めたし、組んでる間はちょー楽しかった。

これからもずっと自作していくことになるなーって気がする。

以上です。
