こちらのサイト [http://d.hatena.ne.jp/murapong/20091207/1260183210:title] を参考にさせていただきました。

何も考えずに、crontab -e を実行したときに、エディタを聞かれたのですが、何も考えずにEnterしちゃって、変なエディタが適用された。(あせった)

で、色々探したが、

>|sh|
$ sudo update-alternatives --config editor
||<

を実行せいと書いてあったが、変更されず…

で、

>|sh|
$ sudo select-editor

Select an editor.  To change later, run 'select-editor'.
  1. /bin/ed
  2. /bin/nano        <---- easiest
  3. /usr/bin/vim.gnome
  4. /usr/bin/vim.gtk
  5. /usr/bin/vim.tiny

Choose 1-5 [2]: 
||<

を実行して、vimを選択して無事変更完了。
