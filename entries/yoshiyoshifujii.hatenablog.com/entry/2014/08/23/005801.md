では、早速、セットアップの儀です。以下の手順だちぇけらー。

1. Raspbianをダウンロード
2. SDカードに書き込む(Macの場合)
3. Raspberry Piを起動する
4. SSHで接続する
5. 初期セットアップをする

====

** 1. Raspbianをダウンロード

以下のサイトからダウンロードしてくる。
[http://www.raspberrypi.org/downloads/:title]

1GBもない。僕はzipをダウンロードした。

[f:id:yoshiyoshifujii:20140823002559j:plain]

** 2. SDカードに書き込む(Macの場合)

僕はMacBook Airなので、Macの手順を参考にしました。
[http://www.raspberrypi.org/documentation/installation/installing-images/mac.md:title]

リンク先に書いてあるとおりですが、

「システム情報」を開きます。
[f:id:yoshiyoshifujii:20140823002858p:plain]

SDカードのBSD名を覚えておきます。(disk1とか2とか)
[f:id:yoshiyoshifujii:20140823002945p:plain]

以下のコマンドを実行します。

>|zsh|
$ sudo dd bs=1m if=2014-06-20-wheezy-raspbian.img of=/dev/disk[n]
||<

ちょっと間かかる…。(30分ぐらいかなー。)

** 3. Raspberry Piを起動する

SDカードに書けたら、取り出して、Raspberry Pi本体に挿します。

ほいで、LANケーブルと、USBのうっすいやつを電源に挿すと、起動!!

電源挿すと即起動やから。

** 4. SSHで接続する

今回は、ディスプレイやらキーボードやらを直接挿さないでしょっぱなから
SSH接続してみる。

まずは、ルーターにアクセスして、DHCPが最近払い出したIPアドレスをおさえよう。

ほいで、IPおさえたら、ターミナルから、

>|zsh|
$ ssh pi@192.168.xxx.xxx
pi@192.168.xxx.xxx's password:
||<

ってなったら、「raspberry」って入力するんや。

ほなアクセスできてまうやろ？

** 5. 初期セットアップをする

こっからは、正直、以下のサイト見てやって問題ないし、見てやって。

[http://news.mynavi.jp/articles/2014/08/13/raspberry-pi/002.html:title]

追加で僕がやったのは、「2 Change User Password」でやす。

初期パスワードのまま置いておくとどっからや奪われてまうかもしらんし。

ほいで、セットアップしたらいつもやってるシステム更新や。

>|zsh|
$ sudo apt-get update
$ sudo apt-get upgrade
||<

を実行するんやー。やー。

あとはそりゃもう、やりたい放題よ。
